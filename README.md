# Predictors-of-childhood-vaccination-uptake-in-England
You can find the implementation of my published journal paper, "Predictors of childhood vaccination uptake in England: an explainable machine learning analysis of regional data (2021–2024)" here

- The original paper is available at: 
https://www.sciencedirect.com/science/article/pii/S0264410X25011995

- You need the COUNTI_1.GEO to visualise the vaccination disparity map (Figure 3 in the paper)
